What is it?

In this project i made an alarm clock, that uses a rgb led strip to wake you up. The alarm clock also lets you set the color of the strip and the intensity. The display i used to show the current time is a graphical display that uses spi as communication.

Why?

I made this project for my studies. But i have always wanted to control my led strips myself, so this was a good opportunity to incoorporate something i wanted to do in a school project.

Things I learned

I learned how to design my own electrical schemes, and design pcb's. I also learned how to use i2c and spi communications. I learned how to solder smd and through hole components. And i learned more about coding.

Things is used

I used Altium designer for designing my pcb. And i used assembly as the programming language. I also used an osciloscope to debug my code.
